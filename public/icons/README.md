# Icon Files Required

This directory needs three PNG icon files:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

Each icon should have:
- Green "S" letter on dark background (#1F2937)
- Simple, clean design matching Savvy Wealth branding

**Note:** Placeholder icons need to be created before building the extension.
